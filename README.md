# Fake store

### Módulo
* `Produtos`

### Api
Este projeto utilizou api https://fakestoreapi.com

### Tecnologias
* `HTML`
* `CSS`
* `SASS`
* `JavaScript`
* `ReactJs`
* `Vite`

### Instalação

#### `yarn`
Download das dependências.

#### `yarn dev`
Rodar o projeto em modo de desenvolvimento.

#### `yarn build`
Cria o aplicativo para produção na pasta 'dist'.
